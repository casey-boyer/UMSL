All programs, invoke 'make'.

To make a specific program from the assignment, invoke 'make P#'.
        EX. To make the executables for problem #3 on the assignment,
            invoke 'make P3'.

To execute a program, invoke './P#' where # is the number of the program
(the problem number on the assignment).

There are currently 3 input images provided in the boyer.2 directory:
	-astronautbw.jpg, the black and white version of astronaut.jpg
	-astronaut.jpg, RGB color image
	-flower.jpg, RGB color image
